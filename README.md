# Conversor de Moedas - Trabalho de Final de Bimestre DDM

Este repositório contém o projeto do aplicativo de conversão de moedas desenvolvido como 
Trabalho de Final de Bimestre para a disciplina de Desenvolvimento de Dispositivos Móveis (DDM).

## Sobre o Projeto

O **Conversor de Moedas** é um aplicativo móvel criado com **React Native**, que permite ao usuário 
converter valores entre diferentes moedas de forma rápida e prática. O app consome uma API 
de câmbio para obter as taxas de conversão em tempo real.

## Funcionalidades

- Conversão de valores entre múltiplas moedas
- Interface intuitiva e responsiva
- Consulta de taxas de câmbio atualizadas online
